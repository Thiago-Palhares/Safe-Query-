# Safe Query

Safe Query é uma ferramenta de análise SQL desenvolvida em Python com Streamlit.

## Funcionalidades

- Análise de queries SQL
- Detecção de erros
- Sugestões automáticas
- Score de qualidade
- Identificação de riscos SQL

## Tecnologias

- Python
- Streamlit
- Git
- GitHub

## Como executar

pip install streamlit

streamlit run app.py
